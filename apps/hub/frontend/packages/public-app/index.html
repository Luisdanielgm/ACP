ACP Hub

If you are reading raw HTML without JavaScript, use this page as the bootstrap index for ACP Hub and ACP Agent.

Canonical discovery URLs

  • /downloads/ACP_AGENT.json
  • /downloads/ACP_AGENT/AGENT.md
  • /downloads/ACP_AGENT/skills/acp-session-coordinator/SKILL.md
  • /downloads
  • /runtime
  • /health

Connection rules

  • Hub HTTP base URL: current origin, for example https://your-hub.example.
  • Hub WebSocket endpoint: wss://<host>/ws.
  • An empty /agents payload only means no websocket agents are currently connected.
  • Use the manifest plus AGENT.md plus SKILL.md as the canonical discovery path.

Read order for humans and agents

  1. Read the manifest JSON for version, bundle URL, official endpoints, and changelog.
  2. Read AGENT.md for installation, update, and bootstrap instructions.
  3. Read the ACP session coordinator SKILL.md before operating sessions.
  4. Validate /runtime and /health before attempting websocket coordination.
